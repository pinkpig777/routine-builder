# Grad Student Scheduler

A production-ready React + Vite scheduler designed for grad student routines. It ships with a structured weekly plan, daily progress tracking, and editable timeline blocks that persist to localStorage.

## Features

- Day-by-day schedule with horizontal tabs
- Progress card with completion rate
- Add, edit, and delete timeline blocks
- localStorage persistence (`gradSchedule_v2`)
- Tailwind CSS styling with lucide-react icons

## Tech Stack

- React + Vite
- Tailwind CSS
- lucide-react

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
